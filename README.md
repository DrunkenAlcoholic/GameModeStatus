# Gamemode Status
Get notified when gamemode is running, This is a basic script that calls "gamemodded -s" for current gamemode status and uses notify-send to notify user

## Requirements
gamemode, python

## Install
```shell
$ chmod +x GamemodeStatus.py
$ GamemodeStatus.py
```
